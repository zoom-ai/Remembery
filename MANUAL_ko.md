# Remembery (리멤버리) — 당신의 삶을 기록하는 디지털 아카이브 미술관

> **Remember (기억하다) + Library (도서관)**
> *“삶의 가장 빛나는 순간들을 영원히 박제하고, 당신의 목소리로 지식을 전하는 품격 있는 유산 보관소”*

---

## Ⅰ. 브랜드 철학 (Philosophy)

인간은 삶을 살아가며 수많은 흔적을 남깁니다. 하지만 시간이 흐르면 기록은 흩어지고 기억은 바랩니다. **Remembery**는 단순한 클라우드 저장소가 아닙니다. 당신의 빛나는 인생 궤적, 지적 탐구의 결과물, 일상에 깃든 사소한 감정까지 한데 모아 **‘생전의 지식 공유’**와 **‘사후의 기품 있는 기념’**을 가능하게 하는 **디지털 헤리티지 아카이브(Digital Heritage Archive)**입니다. 

우리는 소중한 이들에게 남기고 싶은 당신만의 지식과 목소리를 디지털화하여, 세대를 넘어 이어지는 영원의 온기를 제공하고자 합니다.

---

## Ⅱ. 핵심 기능 (Key Features)

### 1. 시네마틱 라이프 타임라인 (Cinematic Life Chronicle)
시간의 강을 따라 물 흐르듯 전개되는 인생의 궤적을 시각화합니다.
* **시간축 자동 정렬**: 기록물 고유의 ‘역사적 날짜(Original Date)’를 기준으로 인생 기록을 정밀하게 타임라인에 배치합니다.
- **동적 연령 계산 (Smart Age Mapping)**: 사용자의 출생 연도를 기준으로 해당 사건 당시의 나이를 실시간 계산하여 표시합니다. (예: `2026년 (48세)`)
- **스크롤 리빌 인터랙션**: IntersectionObserver 기술을 네이티브로 적용하여, 스크롤을 내릴 때마다 과거의 기억이 부드럽게 솟아오르는 몰입도 높은 연출을 선사합니다.

### 2. 하이브리드 사용자 정의 카테고리 (Adaptive Custom Library)
자료의 종류에 맞게 입력창이 유기적으로 변화하는 지능형 스키마를 탑재했습니다.
- **하이브리드 JSON 구조**: 제목, 파일과 같은 공통 메타데이터는 안전하게 검증하며, 논문/일기/사진 등 고유 속성에 필요한 데이터는 데이터베이스 내 유연한 `JSONB/JSON` 포맷으로 유기적으로 수용합니다.
- **카테고리별 맞춤형 동적 필드**:
  - **학술 논문**: ‘저자명’, ‘학술지/저널명’ 등을 기록하는 고급 텍스트 레이아웃.
  - **감성 일기**: ‘오늘 느낀 날씨’, ‘심도 깊은 감정 상태’를 묘사하는 아늑한 레이아웃.
  - **사진/기억**: ‘촬영 장소’, ‘소장 위치’를 안전하게 기록하는 위치 바인딩.

### 3. AI 지능형 조력자 (AI Intelligent Support)
초기 정보 입력의 번거로움을 덜어주는 Gemini AI와 임베디드 파서의 연동 기술입니다.
- **AI 추천 필드 받기 (Gemini Field Suggestion)**: 사용자가 새로운 종류의 카테고리(예: *‘골프 기록’, ‘군대 편지’*)를 만들 때, Gemini API가 이에 꼭 필요한 메타데이터 키와 한글 라벨을 즉석에서 추천하고, 체크박스 선택 하나로 나만의 폼 스키마를 구축합니다.
- **실시간 사진 EXIF 자동 파싱 & 스마트 프리필**:
  - 사진 파일을 선택하는 순간, 백엔드 엔진이 사진 속에 깊이 숨겨진 EXIF 메타데이터를 파싱합니다.
  - **촬영 기기** (Make & Model), **촬영 일시** (DateTimeOriginal), **GPS 좌표** (위도/경도) 정보를 수집하여 즉석에서 우아한 비주얼 칩으로 시각화합니다.
  - 파싱된 정보는 폼 필드의 **기록 당시 날짜** 및 카테고리 내 **촬영 장소** 입력 칸에 스마트하게 **자동 입력(Auto-fill)**되어 완벽한 입력 편의성을 제공합니다.

### 4. 나를 학습한 AI 도슨트 (RAG-based AI Docent)
당신의 기록과 논문, 글을 학습하여 당신의 숨결이 담긴 목소리로 질의응답을 나눕니다.
- **콘텐츠 문자열 합성 (String Synthesis)**: AI가 기록을 분석할 때 커스텀 필드(예: *“이 논문은 [홍길동]이 썼으며 [Nature]에 기재되었습니다.”*)의 고유 텍스트 정보까지 유기적으로 프롬프트에 결합하여 벡터 인덱서에 컨텍스트로 제공합니다.
- **컨텍스트 기반 도슨트**: 대화형 인터페이스를 통해 보관된 자료 전체를 대상으로 고도화된 질의응답(RAG)을 지원하여, 단순한 검색을 넘어 과거의 나와 현재의 나를 잇는 진정한 커뮤니케이션을 지원합니다.

### 5. 가상 테마 전시관 (Thematic Virtual Exhibition Hall)
AI 큐레이터가 보관된 자료를 예술적으로 엄선하여 테마 미술관을 열어줍니다.
- **5대 공간 테마 피팅**:
  - `Classic Museum` (전통 갤러리의 아늑한 황토색)
  - `Burgundy Classic` (풍격이 느껴지는 매혹적인 딥 레드)
  - `Forest Heritage` (수목원의 숨결을 머금은 포레스트 그린)
  - `Ocean Deep` (사색적인 분위기의 바다 청색)
  - `Charcoal Midnight` (작품에 오롯이 스포트라이트를 주는 명암 극대화 다크룸)
- **4가지 공간 레이아웃**:
  - `Docent Flow` (차분하게 서사를 따라 걷는 세로형 관람 방식)
  - `Golden Frame Grid` (황금빛 골드 리프 액자를 씌워 정렬하는 입체 대칭형 격자 방식)
  - `Cinematic Spotlight` (한 작품씩 시네마틱 스포트라이트 조명 아래 감상하는 theatrical 슬라이드 방식)
  - `Bento Collage` (잡지나 현대 미술전처럼 비대칭으로 조화롭게 어우러지는 Bento 격자 방식)

### 6. 안전한 멀티 유저 공간 격리 (JWT)
다른 방문객으로부터 철저히 격리된 비공개 개인 서재를 누리실 수 있습니다.
* **JWT 기반 세션 보안**: 안전하게 암호화된 JSON Web Token(JWT)을 기반으로 로그인 및 회원가입 세션을 관리합니다.
* **Axios 요청 인터셉터**: 프론트엔드에서 나가는 모든 API 호출에 자동으로 토큰(`Authorization: Bearer {token}`)을 포함하여 인증 처리를 단순화합니다.
* **철저한 데이터베이스 스코핑**: 카테고리 생성, 타임라인, 업로드, 가상 전시관 등의 모든 DB 연산은 사용자의 세션 정보(`user_id == current_user.id`)에 종속되어 완벽히 통제됩니다.
* **AI 도슨트 RAG 격리**: 벡터 인덱스를 통한 AI 도슨트 질문 응답 역시 로그인한 사용자가 아카이브한 데이터 조각만을 대상으로 조회하여 타인으로의 데이터 누출을 절대적으로 예방합니다.
* **격리된 프로필 연혁 관리**: 사용자는 연역 밀스톤(경력, 학력, 프로젝트 성과 등)을 직접 추가하고, 본인의 활성화된 프로필에만 국한되는 휴지통 아이콘을 이용해 안전하게 삭제할 수 있습니다.
* **자동 이력서 임포트 타임라인 동기화**: 이력서에서 일괄 저장을 진행할 때, 파싱된 경력/학력/수상/프로젝트 마일스톤이 전용 카테고리 이모지(`🎓`, `💼`, `🏆`, `🌱`)를 사용하여 타임라인 이벤트로 매핑되고 데이터베이스에 직접 연동됩니다.

## Ⅲ. 기술 스택 (Technology Stack)

Remembery는 모던 프론트엔드 아키텍처와 경량화된 초고속 백엔드가 조화를 이루어 작동하는 프리미엄 사양으로 제작되었습니다.

| 레이어 | 기술 | 설명 |
| :--- | :--- | :--- |
| **Frontend Core** | React 19, TypeScript 6.0 | 완벽한 타입 안전성과 고속 렌더링 성능을 자랑하는 차세대 구조 |
| **Styling** | Vanilla CSS, Tailwind CSS | 고품격 뮤지엄 룩앤필(Warm Beige & Ivory)을 위한 미니멀 스타일링 및 세밀한 인터랙티브 모션 효과 |
| **Backend API** | FastAPI, Uvicorn | 비동기 파이프라인 처리를 지원하는 Python 기반 초고속 RESTful API |
| **Database** | SQLite, SQLAlchemy | SQLAlchemy ORM을 매개로 dynamic attributes(JSON) 컬럼을 보관하는 유연한 영속성 구조 |
| **AI Processing** | Gemini API | RAG 질의응답 처리, 맞춤 카테고리 필드 추천 및 전시 자동 도슨트 큐레이팅 제공 |
| **Image Analysis** | Pillow (PIL) | 이미지 바이너리 내 하드웨어 EXIF 태그 파싱 및 GPS DMS 좌표 역변환 연산 모듈 |

## Ⅳ. 사용자 가이드: 6단계 라이프 아카이빙 (User Guide)

당신의 삶을 불멸의 예술로 만드는 Remembery의 단계별 여정입니다.

### 0단계: 회원가입 및 나만의 서재 개설 (User Authentication)
1. Remembery에 처음 접속하면 갤러리 분위기의 우아한 회원인증 포털이 나타납니다.
2. 회원가입(Sign Up)을 눌러 이메일, 비밀번호, 이름을 기입하고 나만의 개인 계정을 생성합니다.
3. 가입된 계정으로 로그인하면, 타인과 철저히 구분된 깨끗한 나만의 대시보드가 준비됩니다.

### 1단계: 나만의 보관 분류 만들기 (Category Setup)
1. 메인 대시보드에서 `새 카테고리 추가` 단추를 누릅니다.
2. 생각하는 기록물의 이름(예: *“어머니의 손편지”*)을 적습니다.
3. **`AI 추천 필드 받기`**를 누르면, Gemini가 편지 발송인, 수신 당시 날씨, 전하고픈 메시지 등 어울리는 필드를 실시간 제안합니다.
4. 마음에 드는 필드를 골라 체크하고 **`생성하기`**를 누르면 끝입니다.

### 2단계: 파일 업로드 및 EXIF 정보 확인 (File Upload)
1. 우상단 `기록물 보관소 업로드` 단추를 누릅니다.
2. 컴퓨터에 소장 중인 옛 사진을 드래그하여 드롭합니다.
3. 하단에 실시간으로 **"사진에서 추출된 정보"** 카드가 나타나며, 촬영 시간, 스마트폰 기종, 정밀 GPS 좌표가 표시되고 해당 정보가 입력 양식의 날짜와 장소 필드에 자동으로 채워집니다.
4. 제목과 자세한 감상을 마저 기록한 후 **`보관소 저장`**을 누릅니다.

### 3단계: 나만의 강물 탐색하기 (Chronicle Timeline)
1. 상단 메뉴에서 `타임라인` 탭을 선택합니다.
2. 당신이 수집한 보관물들이 연도별로 고풍스럽게 흐르는 종적 트랙 위로 스크롤 애니메이션과 함께 등장합니다.
3. 각 연도에 표기된 실제 나이를 확인하며 지나온 삶의 깊이를 음미해 봅니다.

### 4단계: AI 도슨트와 대화하기 (AI RAG Chat)
1. 메인 화면 중앙의 AI 도슨트 질의 응답 창을 켭니다.
2. *"내 기록 중에 1998년에 찍었던 사진 위치가 어디였지?"* 혹은 *"내가 옛날에 쓴 일기들 중 가장 기뻤을 때가 언제야?"* 와 같이 질문합니다.
3. AI는 수동 정보와 사진 EXIF 좌표, 일기의 날씨와 감정 정보를 통찰력 있게 융합하여 당신의 어조로 따뜻하게 응답합니다.

### 5단계: 나만의 전시관 개관하기 (Virtual Exhibition Room)
1. 상단 메뉴의 `전시관` 탭으로 이동합니다.
2. 전시 테마 입력창에 원하는 키워드(예: *“청춘의 여름날”, “나의 학술 여정”*)를 자유롭게 적습니다.
3. 전시 인테리어 색상(`Burgundy`, `Forest` 등)과 공간 배치 기법(`slideshow`, `grid` 등)을 커스터마이징합니다.
4. `전시 큐레이팅 시작` 버튼을 클릭하면, AI가 당신의 데이터를 엄선하여 테마 음악을 튼 듯 우아한 디지털 개인 미술관을 완성해 줍니다.

## Ⅴ. 관리자 및 기여자 가이드 (Developer & Admin Notes)

### 백엔드 실행 방법
```bash
cd backend
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

### Frontend Execution
```bash
cd frontend
npm install
npm run dev
```

### 자동 데이터베이스 마이그레이션 및 격리 검증 테스트
* **스키마 자동 업그레이드**: 백엔드 실행 시 기존 SQLite 테이블 구조를 검사하여 `user_id` 외래키가 누락된 경우 자동으로 추가하고, 기존 아카이브 데이터를 기본 관리자 유저에 안전하게 백필(Backfill)합니다.
* **아키텍처 검증 테스트**: 다중 사용자 및 데이터 격리가 정확하게 작동하는지 보장하기 위해 자동화 테스트 코드가 구현되어 있습니다. 아래 명령어로 즉시 백엔드 인증, 격리 완결성 및 타임라인 동기화 기능을 테스트할 수 있습니다:
  ```bash
  cd backend
  python test_multi_user.py
  python test_auth_isolation.py
  python test_timeline_features.py
  ```
